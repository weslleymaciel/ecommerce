Essa avaliação contempla alguns componentes criados em AngularJS.
Os fontes em typescript estão na pasta "src".

Para iniciar a aplicação será necessário abaixar os arquivos do repositório e acessar "index.html" que está presente na pasta "dist". A aplicação foi testada utilizando o navegador Google Chrome.

OBS: O repositório não disponibiliza todo ambiente para compilação do AngularJS, apenas os arquivos typescript estão presentes.

Não foi implementado o backend, desta forma os produtos cadastrados estão sendo inicializado no componente inicial. Não há movimentação em um banco de dados.
